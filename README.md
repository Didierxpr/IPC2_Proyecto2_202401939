# IPC2 Proyecto 2 - 202401939

Este proyecto se implementa con Razor Pages en C# y usa estructuras propias sin colecciones genericas para la logica principal.

## Release 1

- Inicializacion del sistema en memoria.
- Gestion de drones:
  - Agregar dron validando nombre unico.
  - Listar drones en orden alfabetico.
- Implementacion de TDA propio:
  - Lista enlazada simple para drones.

## Requisito de entorno

Para ejecutar el proyecto se necesita instalar el SDK de .NET 8.

Comandos esperados una vez instalado:

```powershell
dotnet restore
dotnet run
```
